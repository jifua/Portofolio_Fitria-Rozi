# Output Visualisasi

Folder ini berisi hasil visualisasi dari proses analisis data e-commerce.

Setiap grafik dibuat menggunakan Python (Matplotlib) untuk mendukung interpretasi data dan pengambilan keputusan berbasis data.

---

## 📁 Daftar Output

* **price_distribution.png**
  Distribusi harga produk untuk melihat pola sebaran harga.

* **rating_vs_recommendation.png**
  Hubungan antara rating produk dan probabilitas rekomendasi.

* **sentiment_analysis.png**
  Pengaruh sentiment review pelanggan terhadap rekomendasi produk.

* **brand_analysis.png**
  Perbandingan brand berdasarkan rata-rata rating produk.

* **season_analysis.png**
  Analisis pengaruh musim terhadap rekomendasi produk.

* **price_segment.png**
  Analisis performa produk berdasarkan kategori harga.

* **top_products.png**
  Produk terbaik berdasarkan skor kombinasi (rating + sentiment).

* **correlation.png**
  Korelasi antar variabel untuk memahami hubungan antar fitur.

---

## 📌 Catatan

Seluruh visualisasi digunakan untuk mendukung insight bisnis dan membantu dalam pengambilan keputusan strategis.
