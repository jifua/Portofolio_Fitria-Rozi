# Data Cleaning & Preprocessing (Python)

This module handles data cleaning and preparation before storing into the database.

## Workflow

Load → Clean → Transform → Validate → Export

## Key Processes

* Handled missing values and inconsistent records
* Standardized categorical variables (supplier, category)
* Normalized numerical features (quality & efficiency)
* Ensured data consistency and integrity
* Exported clean dataset for SQL analysis

## 📂 File

* `cleaning_sqlite.py` → Main data preprocessing script

## Output

* Clean dataset ready for database and analysis

## Purpose

Ensures that raw production data is reliable and usable for downstream analysis and dashboarding.
