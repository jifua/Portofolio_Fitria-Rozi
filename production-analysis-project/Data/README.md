# Data Description

This folder contains both raw and cleaned datasets used in the project.

## Files

* `production_data.csv` → Raw dataset
* `Clean_Production_Data.csv` → Cleaned dataset after preprocessing

## Data Overview

The dataset simulates production process data, including:

* Production date
* Supplier
* Mixing speed
* Pigment type
* Product quality score
* Efficiency score

## Purpose

* Raw data → preprocessing
* Clean data → SQL & Power BI analysis

## Note

The cleaned dataset has been processed to ensure:

* Consistency
* No missing values
* Ready for analysis
