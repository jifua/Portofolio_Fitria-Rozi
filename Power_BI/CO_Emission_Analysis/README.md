# 🚗 CO2 Emission Analysis Dashboard

## Business Problem
Perusahaan otomotif menghadapi tekanan untuk mengurangi emisi CO2 akibat regulasi lingkungan dan tuntutan sustainability.  
Diperlukan analisis untuk mengidentifikasi faktor utama yang mempengaruhi emisi kendaraan.

---

## Objective
- Mengidentifikasi faktor utama penyebab emisi CO2
- Membandingkan performa berbagai jenis kendaraan
- Mendukung strategi pengurangan emisi

---

## Tools
- Power BI
- Dataset kendaraan (Cousera)

---

## Data Understanding
Dataset mencakup:
- Engine Size
- Fuel Type
- Powertrain
- Transmission
- CO2 Emissions

## Key Insights
- Engine size memiliki pengaruh paling besar terhadap emisi
- Kendaraan berbasis Internal Combustion Engine (ICE) menghasilkan emisi tertinggi
- Fuel type petrol memberikan kontribusi emisi lebih besar dibanding hybrid/electric
- Powertrain hybrid menunjukkan efisiensi lebih baik

---

## Business Solution
- Mengurangi produksi kendaraan berbasis ICE
- Mengembangkan kendaraan hybrid & electric
- Optimalisasi ukuran mesin (engine downsizing)
- Strategi pemasaran berbasis eco-friendly

---

## Impact
- Penurunan emisi karbon
- Efisiensi bahan bakar meningkat
- Mendukung regulasi lingkungan
- Meningkatkan citra perusahaan (green industry)

---
