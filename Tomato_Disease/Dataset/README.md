# Dataset – Tomato Leaf Disease

## 📌 Deskripsi

Dataset digunakan untuk melatih model klasifikasi penyakit daun tomat yang terdiri dari 10 kelas (9 penyakit + 1 healthy).

---

## Struktur Dataset

* Total awal: 18.834 gambar
* Setelah balancing: 8000 gambar
* Per kelas: 800 gambar

---

## Pembagian Data

* Training: 70%
* Validation: 15%
* Testing: 15%

---

## Preprocessing

* Resize: 224x224
* Normalisasi pixel
* Data augmentation:

  * Rotasi
  * Zoom
  * Brightness

---

## Insight 
* Dataset awal tidak seimbang → berpotensi bias model
* Dilakukan balancing untuk meningkatkan generalisasi
* Variasi data penting untuk meningkatkan performa model

---

## Sumber Dataset

Mendeley : J, ARUN PANDIAN; GOPAL, GEETHARAMANI (2019), “Data for: Identification of Plant Leaf Diseases Using a 9-layer Deep Convolutional Neural Network”, Mendeley Data, V1, doi: 10.17632/tywbtsjrjv.1

---

## Value

Dataset ini menjadi fondasi utama dalam membangun model AI yang akurat dan robust terhadap variasi kondisi.
